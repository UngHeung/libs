# 기능 라이브러리 만들기

## 필요 라이브러리 설치

### 프로젝트 생성

```powershell
npx create-react-app@latest --template typescript
```

### react-router-dom

```powershell
npm i react-router-dom
```

### styled-components

```powershell
npm i styled-components
npm i -D @types/styled-components # 타입 정의
```
